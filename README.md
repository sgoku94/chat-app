# 실시간 채팅 애플리케이션

React와 Laravel을 사용한 실시간 채팅 애플리케이션입니다.

## 주요 기능

- 회원가입 및 로그인
- 실시간 채팅
- 1:1 채팅 및 그룹 채팅
- 채팅방 나가기 및 초대
- 친구 관리 (추가/삭제)
- 메시지 읽음 표시
- 실시간 알림
- 반응형 디자인

## 기술 스택

### Frontend
- React
- React Router
- Axios
- Pusher.js (실시간 통신)
- CSS Modules

### Backend
- Laravel
- Laravel Sanctum (인증)
- Laravel Pusher (실시간 통신)
- MySQL
